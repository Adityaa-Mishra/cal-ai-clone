# Cal AI Clone

AI-powered calorie tracking application built for the 8x Social Build Challenge.

## Features

- AI food recognition from photos
- Calorie estimation
- Protein, carbs, and fat tracking
- Daily food diary
- Weekly analytics charts
- Streak tracking
- Goal management

## Tech Stack

- React Native
- Expo
- Gemini Vision API
- AsyncStorage / SQLite

## Screens

- Home Dashboard
- Food Scanner
- Nutrition Results
- Daily Diary
- Weekly History
- Streak Tracker
- Settings

## AI Logs

Development logs are stored in the `/ai-logs` folder.
